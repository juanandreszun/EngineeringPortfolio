Project Goal

Write code to create an SVG file for a Laser Cutter

Step 1

Sketch out the dimensions you want for your part. In this project I made boxes.

Step 2

Write some code in Matlab that opens a file and puts user defined dimensions in an SVG readable format.

Step 3

Open your text file as an SVG to see the shape that it traces.

Step 4

Import your SVG into your laser cutter for laser cutting.

Step 5

Put your favorite parts in your newly formed boxes after adding screws to hold the panels together.
